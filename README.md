# WechatUnrecalled 

Prevent Wechat recalling chat messages and comments from sns (moments).

防止微信撤回聊天消息和朋友圈回复

支持金色共存版，LBE平行空间（双开大师）

选择性的支持 6.0.0, 6.3.5, 6.3.8, 6.3.9

[download/下载](https://github.com/fkzhang/WechatUnrecalled/releases)
